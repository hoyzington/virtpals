# virtpals

Welcome to virtpals! A bad business idea, but a fun Sinatra web app project.

## Installation

Clone this app and then execute:

    $ bundle install

## Usage

Enter `shotgun` into the CLI. Then go to `http://127.0.0.1:9393/`
